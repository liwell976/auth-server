# Auth Server – TP1 à TP4

Serveur d'authentification REST sécurisé en Java avec Spring Boot.

## Stack technique
- Java 17
- Spring Boot 3.x
- Maven
- MySQL (production) / H2 (tests)
- Client : JavaFX

## TPs
- TP1 : Authentification dangereuse (mot de passe en clair)
- TP2 : Authentification fragile (BCrypt + anti brute force)
- TP3 : Authentification forte (HMAC + nonce)
- TP4 : Master Key + CI/CD
